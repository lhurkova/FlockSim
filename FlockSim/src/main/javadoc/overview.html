Short overview of the FlockSim

FlockSim is a simulation enviroment used for agent-based flocking simulations. Application is based on plug-ins that can be loaded and used for the simulation. Plug-ins represents flocking models with different behavior and parameters. FlockSim has a graphical user interface and displays graphically the course of the simulation.

Simulation proces

After loading a plug-in simulation can be run. First specified number of agents (representing birds) are initialized with random positions and random velocity vectors. Than a specified number of steps is executed. During each step of the simulation all agents execute their steps and graphical component is repainted.

Program structure

Main class of FlockSim is FlockSim. FlockSim initialize the graphical user interface represented by GUI class. GUI uses the FlockGraphics class to graphically display the course of the simulation and classes AboutDialog and Settings to dispaly about a settings dialogs. Class Point represents a point in Euclidean space and provides basic geometric operations with points and vectors. Class Point2D representa a point in 2D space.

Class Flock represents a flock during the simulation. It contains Agents that are stepped during each step of the simulation. FlockModel is an interface that defines methods for creating Agents and getting informatin about specific implementations of a Agent interface. AreaPositions class is a supporting class for Flock that help with finding agents in specified areas.

Parameters of FlockModel and Agents can be described by classes extending the class Descriptor such as ComboBoxDescriptor, IntFieldDescriptor and SliderDescriptor. Each descriptor has its grafical representation implemented by classes ComboBoxItem, IntFieldItem and SliderItem that all extend SettingsItem class. Settings items are graphically displayed in settings dialog and values from them inputed by the user are storen in Parameters class.
